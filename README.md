# Overview
This repository includes projectfiles used from OpenSeaMap project for generation of offline charts.

The files are available in MOBAC and/or OSMCB  formats.

Mobac - Mobile Atlas Generator 
OSMCB - Open Sea Map Chart Bundler

# atlas/osmcb/sea
Several projectfiles for sea areas. 
The files where created from OSM member.

# atlas/osmcb/inland-waterways
Several projectfiles for european inland waterways
The files where derived from offline maps downloaded from sourceforge:
https://sourceforge.net/projects/opennautical
